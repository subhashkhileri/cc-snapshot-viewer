# Changelog

All notable changes to the CC Snapshot Viewer extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-12-30

### Added

- Initial release
- View file changes per Claude Code prompt in the Source Control tab
- Side-by-side diff view for modified files
- Compare any snapshot with the current file state
- Auto-refresh when Claude makes new changes
- Multi-workspace support
- File watcher with debouncing for efficient updates
